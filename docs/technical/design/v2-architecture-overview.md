# V2 Architecture Overview

This document records the architectural direction that is now mostly implemented in the runtime.

## Implemented Foundation

- CLI command surface.
- Query normalization.
- Tool planning.
- Staged execution.
- Raw/extraction/derivation/claim effects.
- DuckDB-backed storage indexes.
- Handler versioning.
- Claim/triple projection for core handlers.

## Remaining V2 Work

- Better evidence inspection in CLI output.
- Semantic reduction over claims.
- Learner-facing semantic display.
- Optional hydration.
